## Interactive Elements

- **Find Device Button**: Primary blue background, white text
- **Disconnect Button**: Red background, white text
- **Device List Items**: Light gray background, tap feedback
- **Close Button**: Icon in circular badge background
- **Status Badge**: Colored pill with white text

## Loading States

- **Scanning**: Spinner animation + "Scanning..." text
- **Connecting**: Spinner animation on device item
- **Disconnecting**: Spinner animation on button

## Error States

- **Connection Failed**: Red alert popup with error message
- **Permission Denied**: Alert directing to settings
- **Scan Error**: Logged to console, user shown "No devices found"

## Accessibility

- All buttons have clear labels
- Status indicators are color + text (not color alone)
- Device IDs are truncated but full on tap
- Loading states clearly indicated with text

**This UI is fully responsive and works on phone and tablet screens.**
